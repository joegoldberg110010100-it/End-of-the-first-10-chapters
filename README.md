# Rust Book – Chapters 1–10

This is a small practice project I created after finishing the first **10 chapters** of **"The Rust Programming Language"** (also known as *The Rust Book*).

The goal of this project is to combine and practice the concepts I learned, including:

- Modules
- Structs
- Methods
- Ownership
- Borrowing
- References
- Lifetimes
- Pattern matching
- `Option` and `Result`
- Basic project organization

The project is intentionally simple. It serves as a personal milestone and a reminder of how much I had learned after completing the first part of the book.

## Why I made this

I wanted to build something on my own instead of only reading examples from the book. While it's a small project, it represents an important step in my Rust learning journey.

More projects will come as I continue learning Rust.

🦀 Thanks, Rust Book!
